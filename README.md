# Shopverse-ecommerce-ui

A responsive Ecommerce UI built using HTML, CSS, and JavaScript. The project provides a modern shopping interface with product search, shopping cart functionality, dark mode, and an automatic hero image slider.

## Features

- Responsive Ecommerce UI
- Dynamic Product Rendering
- Product Search Functionality
- Shopping Cart with LocalStorage
- Dark Mode Toggle
- Hero Image Slider
- Hamburger Navigation Menu
- GitHub Deployment

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- LocalStorage
- Git & GitHub

## How It Works

### Shopping Cart
- Users can add or remove products from the cart.
- Cart data is stored in LocalStorage.
- Cart count remains saved after page refresh.

### Search Feature
- Users can search products using the search bar.
- Products are filtered dynamically.

### Dark Mode
- Users can switch between light and dark themes.

### Hero Slider
- Hero section images automatically change at regular intervals.
